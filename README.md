# JSON-Log-Country
The "combine-JSON.sh" file is an easy way to combine a large number of JSON files from a single directory into a single file.
This script requires the jq (jquery utility to be installed)
    - JQ github: https://github.com/jqlang/jq
    - Debian: sudo apt install jq
    - RPM (CentOS, Fedora): sudo yum install jq
    - DNF (Fedora => v22): sudo dnf install jq
    - Arch -= sudo pacman -S jq


Working a python script for various lookups and subsequent graphical data analysis
